全局变量模块

获取变量、读取变量

全局变量模块 对脚本的参数提供了读取和写入作为寄存

核心 API 详解

_G[key]
实际上就是 ConcurrentDictionary data
参数名 类型 描述 必填
key string 指定变量名词 是
返回值:

object: 表示操作结果值 var 、const

返回值 含义
int 整数
double 浮点数
bool 布尔

完整使用示例

// 设置数据
_G["int"] = 1234;
// 获取数据
var result = _G["int"]

使用注意事项

  • 在使用 GetNextImage() 前，必须先调用 ReadFileImageList() 加载图像列表
  • 所有文件路径应使用绝对路径或相对于应用程序的正确相对路径
  • 保存图像时，文件扩展名决定保存格式，支持的格式包括：.bmp, .png, .jpg/.jpeg, .tiff
  • 处理大图像时请注意内存使用，及时释放不再需要的图像对象
  • 当方法返回错误时，详细错误信息会被记录到系统日志，可通过日志系统查看
